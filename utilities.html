IBMCloudAccess Class¶

The module provides IBMCloudAccess class, and the following module-level utilities:

  • rename_keys(): Renames keys in a dict.

  • static_vars(): Decorator that assigns initial values to ‘static’ variables in function. To access these variables, use <func_name>.
